<h1 style="align: center;">Tiktok Downloader</h1>
<h3>it's tiktok downloader with mass or single download options </h3>
<h2>Installation</h2>
run in your terminal:

```
git clone https://github.com/karim0sec/tiktokdl
cd tiktokdl
npm i
```
<h2>usage</h2>

npm run tiktokdl
